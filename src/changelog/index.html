Changelog
Editor, layouts and powerv0.4
  • Full screen text editor with save and quit shortcuts
  • Seven selectable keyboard layouts
  • Command history with up and down recall
  • calc, logo, and ASCII boot art
  • shutdown and reboot via QEMU power ports
Disk persistencev0.3
  • Hand written ATA PIO driver (primary, master)
  • CMFS: a custom on-disk filesystem format
  • Files now survive a reboot
A real shellv0.2
  • In-memory filesystem: ls, cd, mkdir, cat, touch
  • PC speaker driver and sound commands
  • Five colour themes
First bootv0.1
  • Multiboot2 and GRUB boot chain
  • VGA text mode driver
  • Polling PS/2 keyboard input
